⚠️ ATENÇÃO: vai substituir TODOS os dados atuais pelos do arquivo. Faça um backup antes se tiver dúvida.
Pra enviar seus backups direto pro Google Drive, siga este passo a passo:
  1. Clique em 💾 Baixar arquivo de backup acima
  2. Abra drive.google.com no navegador
  3. Crie uma pasta chamada JP Embalagens Backups (uma vez só)
  4. Arraste o arquivo mtx_backup_*.json pra dentro dessa pasta
  5. Pronto! Faça isso 1x por semana
💡 Dica: se você usar Drive instalado no PC, é só baixar pra pasta dele e sincroniza sozinho.
